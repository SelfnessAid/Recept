Fungerar inte länken?

Du måste använda Safari-webbläsaren på din iPhone för att det ska fungera.

Detaljerade instruktioner

Klicka på länken, välj att ladda ner appen, stäng därefter webbläsaren och hitta appen på din hemskärm. Klicka på appen och välj "Lita på" om telefonen frågar om du vill lita på utvecklaren för appen.
